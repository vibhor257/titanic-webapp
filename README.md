# titanic-webapp

## Installation
### Clone the repo
- `git clone <>`
- `cd titanic-webapp/src`

### Local Setup
- Step 1: Create py virtual env
    - `python3 -m venv py_env_titanic`
- Step 2: Activate Env
    - `source  py_env_titanic/bin/activate`

- Step 3: Install requirements
    - `pip3 install -r requirements.txt`
